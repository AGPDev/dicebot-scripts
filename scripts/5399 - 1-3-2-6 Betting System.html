How do I use this?

  1. Click the View script Button while viewing a script.
  2. Open your bot.
  3. Set the settings mode to programmer mode.
  4. Paste the script above into the code box.
  5. Go to the console tab.
  6. Type nextbet = xx.xxxx and press enter in the input box at the bottom, where xx.xxxx is the amount of your first bet.
  7. Type start() and press enter to start the bot, and stop() to stop to the bot. (you need to be logged in before starting the bot)

See The programmer mode page for more information on the programmer mode

How 1-3-2-6 System works:

Step 1 won  - You win your 1-unit bet. The next bet will be x3 of 1st bet.
Step 2 won  - The next bet will be x2 of 1st bet.
Step 3 won  - The next bet will be x6 of 1st bet.
Step 4 won  - You win and start over.

Step 1 loss – You lose your 1-unit bet. The next move is to start over.
Step 2 loss – You lose and start over. If you lose at step 2 you lose 2 units, as the remaining unit was profit from the last step.
Step 3 loss – You lose and start over. If you lose here, you’ll have walked away with a 2-unit profit.
Step 4 loss – You lose and start over.
