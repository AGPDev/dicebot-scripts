How do I use this?

  1. Click the View script Button while viewing a script.
  2. Open your bot.
  3. Set the settings mode to programmer mode.
  4. Paste the script above into the code box.
  5. Go to the console tab.
  6. Type nextbet = xx.xxxx and press enter in the input box at the bottom, where xx.xxxx is the amount of your first bet.
  7. Type start() and press enter to start the bot, and stop() to stop to the bot. (you need to be logged in before starting the bot)

See The programmer mode page for more information on the programmer mode

This script is inspired by the "c49" script: https://bot.seuntjie.com/scripts.aspx?id=2131 (not mine)
I've modified it and added statistics and clock.

Sorry for my bad english, please correct some terms if necessary.

Minimum balance recommended: 50000 satoshis.

The principe is simple: all bets are equals to basebet, and the script cumulate the bets amount since a lose streak until a winning bet (betstack).
You are in profit after 2 consecutives winning bets and script restart from basebet.

I've add a multiplier for the "betstack", so you can adjust your settings by editing basebet, chance or multiplier to make more profit, but it's more risked.
It depends of your balance, your luck, and your "cojones" :)

You can use the statistics to refine your settings:
-->BETSTACK= Amount of bets accumulated until the next winning bet.
-->BESTWINBET= Best winned bet.
-->LONGSTREAK= Longuest streak without profit.
-->WORSTLOSEBET= Worst losed bet.
-->WORSTLOSESTREAK= Worst balance falling.

-The statistics are in satoshis, but you can change it easily if you want.

-The seed's changing every 1000 bets if it's allowed by the website.

-Betting high/low is random.


Don't forget, there is no "magic script", use it at your own risk, don't use it too long time.
